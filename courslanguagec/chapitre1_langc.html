Chapitre 1 : Introduction au Langage C

Qu'est-ce que le langage C ?

Le langage C est un langage de programmation impératif, structuré et très utilisé. Il a été créé au début des années 1970 par Dennis Ritchie. Il est connu pour sa rapidité d'exécution et sa proximité avec le matériel, ce qui le rend idéal pour le développement de systèmes d'exploitation, de logiciels embarqués, de pilotes, etc.

Pourquoi apprendre le C ?

  • Base pour comprendre d'autres langages comme C++, Java, etc.
  • Exécution rapide
  • Utilisé pour la programmation système
  • Très répandu dans les systèmes embarqués

Premier programme en C

Voici un exemple simple d’un programme en C qui affiche « Bonjour le monde » :

#include <stdio.h> int main() { printf("Bonjour le monde !\n"); return 0; }

Explication :

  • #include <stdio.h> : Inclut la bibliothèque standard d’entrée/sortie.
  • int main() : Point d’entrée du programme.
  • printf() : Fonction qui permet d’afficher du texte.
  • return 0; : Indique que le programme s’est terminé avec succès.
