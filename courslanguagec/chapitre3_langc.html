Chapitre 3 : Structures conditionnelles en C

1. Structure conditionnelle simple : if

if (condition) {
    // instructions si la condition est vraie
}

2. if...else

if (condition) {
    // instructions si la condition est vraie
} else {
    // instructions si la condition est fausse
}

3. Conditions multiples : if...else if...else

if (condition1) {
    // instructions
} else if (condition2) {
    // instructions
} else {
    // instructions
}

4. Opérateur ternaire (facultatif)

(condition) ? valeur_si_vrai : valeur_si_faux;

Exercice d'application

Écrire un programme en C qui demande à l’utilisateur de saisir un nombre entier et affiche s’il est pair ou impair.

Solution :

#include <stdio.h>

int main() {
    int nombre;
    printf("Entrez un nombre entier : ");
    scanf("%d", &nombre);

    if (nombre % 2 == 0) {
        printf("Le nombre est pair.\n");
    } else {
        printf("Le nombre est impair.\n");
    }

    return 0;
}
