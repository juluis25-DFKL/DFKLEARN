Chapitre 2 : Variables et Types en C

1. Déclaration de variables

Une variable est un espace mémoire nommé utilisé pour stocker une valeur. En C, on doit toujours déclarer le type de la variable avant de l'utiliser.

int age; float taille; char lettre;

2. Types de base

  • int : entier (ex : 3, -12)
  • float : réel à virgule flottante (ex : 1.5, 3.14)
  • char : un caractère (ex : 'a', 'Z')

3. Affectation et initialisation

On peut affecter une valeur à une variable avec =.

int age = 25; float note = 18.5; char initiale = 'J';

4. Affichage et saisie

printf("Âge : %d", age); scanf("%d", &age); // saisie d'un entier
