Chapitre 5 : Les Tableaux

1. Déclaration et remplissage

Un tableau est une structure qui permet de stocker plusieurs valeurs dans une seule variable. On peut déclarer un tableau comme ceci :

tableau[5]

Cela crée un tableau de 5 éléments. On peut remplir un tableau comme suit :

Pour i allant de 0 à 4 faire
    Saisir tableau[i]
FinPour

2. Parcours avec boucles

Pour accéder aux valeurs d’un tableau, on utilise une boucle :

Pour i allant de 0 à 4 faire
    Afficher tableau[i]
FinPour

On peut aussi faire des calculs comme la somme des éléments :

somme ← 0
Pour i allant de 0 à 4 faire
    somme ← somme + tableau[i]
FinPour
Afficher somme
